# arm-templator-usage
Usage example for https://github.com/anthony-c-martin/arm-templator

## Getting Started
1. Clone this repo
2. Run `npm install`
3. Run `npm start {fileName}` where 'fileName' is the name of the template in the root folder

## Sample Templates

### [Basic](./basic.ts)
A basic starting point for template generation.

Run `npm start basic`
